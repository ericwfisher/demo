# Demo

Clone the repository to a directory of your choice

```
git clone https://github.com/ericwfisher/demo.git
```

### Install

```
npm i
```

### Lint

```
npm run lint
```

### Unit Test

```
npm run test
```

### Serve
To just run the app
```
npm run serve
```

For development
```
npm run devserve
```
